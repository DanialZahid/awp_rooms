# Rooms (awp_rooms) - README

## Description

Rooms (awp_rooms) is a Sniper War scenario level for Counter-Strike: Global Offensive.\
It's a redesign of my Counter-Strike level [awp_rooms](https://gamebanana.com/maps/196214).

---
